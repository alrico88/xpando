# Xpando

> Unshorten any URL.

This website is built with [Next.js](https://nextjs.org/).

You can access the page [here](https://xpando.vercel.app) or use the API `https://xpando.vercel.app/api/resolve?site=YOUR_URL_HERE`.

## Try it out locally

### Setup

```bash
# install dependencies
corepack enable
pnpm install

# serve in dev mode
pnpm dev
```
